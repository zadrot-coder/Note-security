# 🌟 Консольная система заметок (Notes CLI)

Простая, быстрая и удобная консольная записная книжка, написанная на <Python> с использованием встроенной базы данных **SQLite**.

🚀 Возможности
- 📝 Раздельный ввод обычных заметок, долгов и дней рождения.
- 🗄️ **Вечное хранение**: данные записываются в файл базы данных `my_notes.db` и не стираются после закрытия программы.
- 🎯 Автоматическая сортировка записей по категориям.
- 🔍 Интерактивный просмотр сохраненных данных по отдельным категориям.

🛠️ Технологии
- Python 3.x
- SQLite3 (встроенная реляционная база данных)
- Time (эффекты плавной загрузки данных)

📦 Как запустить проект
1. Склонируйте репозиторий или скачайте файл `my_notes.py`.
2. Запустите программу через терминал или среду разработки:
   ```bash
   python my_notes.py
   ```
3. Для работы не требуется установка сторонних библиотек (все модули встроенные).

🗂️ Архитектура кода
В основе проекта лежит класс `Notes`, который инициализирует объекты карточек. База данных автоматически создается при первом запуске приложения с помощью функции `init_db()`
